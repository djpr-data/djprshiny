Explanation

This page compares options for embedding the VIC font in CSS for websites:

VIC (system)
Uses the system-installed font named "VIC". This will fall back to something else if the font is not installed — as it won't be for the vast majority of Victorians.
VIC-font
Multiple @font-face declarations pointing to separate files for variants. The files come from the official download.
VIC-Regular
Uses only VIC-Regular font file and relies on embedded/browser-generated variants.
VIC-Regular/VIC-Bold
Uses VIC-Regular for normal weight, VIC-Bold for bold weights with bolding turned off in styles. Relies on embeded/browser-generated variants for italics. This is what the DJPR public website does (at time of writing), so may be considered a source of truth.

Obsevations:

  • The system-installed fonts appear lighter than the official web fonts, as used by the DJPR public website.
  • VIC-Regular's bold variant (created by the browser) is light compared to the official Vic-Bold font.
  • Using VIC-Regular/VIC-Bold would require switching font families to get different weights, which does not match common practice. Headings will be too heavy unless care is taken to reset bolding no all headings in CSS.
